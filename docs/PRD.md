# PRD: VOE Listening Platform (MVP)

## 목적
한국어 기반 직원 의견(Voice of Employee)을 조직 맥락 안에서 안전하게 수집하고, 반복 신호를 구조화해 실행 가능한 인사이트로 전환한다.

## 문제 정의
- 의견 수집은 많지만, 조직 맥락 없이 흩어져 실행으로 이어지지 않는다.
- 익명성과 심리적 안전이 부족해 참여가 낮다.
- HR/리더가 빠르게 패턴을 발견하고 행동으로 연결하기 어렵다.

## 목표
1. 직원이 안전하게 의견을 제출할 수 있는 신뢰 경험 제공
2. HR이 조직 단위로 의견을 해석하고 우선순위를 세울 수 있는 구조 제공
3. 리더가 실행 계획과 연결할 수 있는 요약/인사이트 제공

## 핵심 사용자
- 일반 직원(의견 제출자)
- HR/관리자(패턴 분석 및 실행 담당)
- 리더(의사결정 및 우선순위 설정)

## MVP 범위
- 랜딩 페이지, 조직 보드, 조직 보드 상세, 의견 제출, 관리자 대시보드, 인사이트 페이지
- 조직 트리 기반 탐색 및 상하위 조직 집계
- 텍스트/오디오 제출(오디오 전사는 모의 처리)
- 조직 필터링 및 롤업 집계
- 한국어 UI/UX 및 신뢰 중심 카피

## 주요 사용자 흐름
1. 직원: 랜딩 → 조직 보드 → 의견 제출
2. HR: 대시보드 → 조직 필터 → 인사이트 확인 → 액션 카드 확인

## 성공 지표 (MVP)
- 의견 제출 전환율
- 익명 제출 비중
- 조직 보드 방문/체류
- 인사이트 확인 후 액션 카드 생성/진행 비율

## 제외 범위
- 실제 AI 모델 연동(현 단계는 인터페이스/모의 서비스)
- 실제 오디오 전사/파일 저장
- 권한/SSO/감사 로그
